Volant

A volunteer placement tool for organizations such as SCI or those united under the Alliance Network. Developed for (INEX-SDA)[http://www.inexsda.cz/en] but can be any similar NGO with similar workflow.

Built on Rails 4 and Ember.js

Installation on-premise

Requirements

Those tools need to be installed on top of a normal Debian/Ubuntu server:

  • postgres tools

    sudo apt-get install libpq-dev

  • wkhtmltopdf (for PDF versions of VEFs)

    sudo apt-get install wkhtmltopdf xvfb echo 'xvfb-run --server-args="-screen 0, 1024x768x24" /usr/bin/wkhtmltopdf $*' | sudo tee --apend /usr/local/bin/wkhtmltopdf
